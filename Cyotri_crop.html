Cyrotri Crop

Cyrotri Crop is a multinational company that manufactures consumer based electronics and equipment.Electronic Components are intended to be connected together, usually by soldering to a Printed Circuit Board (PCB), to create an electronic circuit with a particular function (for example an amplifier, radio receiver, oscillator, wireless). Some of the Main Electronic Components are: Resistor, Capacitor, Transistor, Diode, Operational Amplifier, Resistor Array, Logic Gate etc.
